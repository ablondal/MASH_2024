Dinosaur DNA

Alex just landed her dream job, an internship at InGen on the beautiful Isla Nublar! She's sequencing dinosaur DNA retrieved from ancient mosquitoes trapped in amber. In order to analyze the complexity of the DNA she's studying, she decomposed the DNA into a set of base strands, strands of DNA from which all of the dinosaur DNA can be composed (through concatenation). Note that no base strand can be made by concatenating other base strands, since then it wouldn't be a base strand.

Yesterday, she made samples of each base strand, and put them in seperate test tubes. When she came into work today, however, all of the test tubes had been poured into a single basin, and all the base strands were mixed together! She's retrieved all the strands of DNA from the mixture, but is unsure if any of the base strands have combined to form new strands. Help her figure out whether this is the case!

Input

The first line of the input consists of a single integer, $n$, the number of strands that Alex has retrieved.

Following are $n$ unique DNA strands $s_1, \dots, s_n$, each a string consisting only of the letters ACGT.

Output

If there exist any composite strands, that is, strands formed by concatenation of other strands, output a line containing Yes. Otherwise, output a line containing No.

If there were composite strands, illustrate one of them by showing how it is formed by concatenating other strands. See Sample 1 for an example of the format.

Bounds

$\sum_{i=1}^n |s_i| \leq 50000$

Sample 1

stdin

4 AT CG CGAT ATAT

stdout

Yes CG + AT = CGAT

Note that AT + AT = ATAT would also have been a valid answer.

Sample 2

stdin

3 AGT GTA TCG

stdout

No
